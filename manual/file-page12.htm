Tasks

 

 

See chapter 15 how to add labels.

The Status field is set to Open by default for a newly created task (see 07). Another entry can be selected from the pull down when clicking on the Status field.

A document can be attached to a task by either clicking on the NEW ATTACHEMENT(S) field (see 09) or dragging a document from a local directory on your PC into this field. When clicked on, a dialog panel will open to enable you to naviagte to the appropriate file location and select the file to upload. For more details see chapter 15.

The task will be activated by clicking the SAVE button (see 10).
